HTTP API

Use Atoqu as a local HTTP search service

Start HTTP Server

./atoqu --http 8080

/search

curl -X POST http://localhost:8080/search \
  -H "Content-Type: application/json" \
  -d '{"query": "atomic search engine"}'

/feedback

curl -X POST http://localhost:8080/feedback \
  -H "Content-Type: application/json" \
  -d '{"id": "doc1", "mode": 0, "score": 1.0, "query": "atomic search engine"}'

/stats

curl http://localhost:8080/stats
